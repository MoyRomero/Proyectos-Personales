PORTAFOLIO

Recopilación de proyectos (Sitios Web, Aplicaciones Web (PWA), Web APIs) personales, realizados con el fin autodidáctico, con tecnologías: .NET, C#, SQL, HTML, CSS y JavaScript.

SISTEMA DE RESERVA DE VIAJE

Sistema de reserva de viaje de autobús, el cual tiene la capacidad de realizar reservas para viajes entre distintos destinos y con fechas determinadas.

Los datos se almacenan en una base de datos local cuyo motor es MSSQL.

Tecnologías usadas: ASP.NET, C#, SQL, HTML, CSS, JavaScript.

TO DO LIST

Aplicación Web (PWA) cuyas principales funciones es almacenar texto como tareas (o notas personales), sin la implementación de Bases de datos o motores de bases de datos, sino mediante el guardado de información JSON, en LocalStorage programado con JavaScript.

Tecnologías usadas: HTML, CSS, JavaScript.

CSS CURSE

Sitio Web desarrollado con el fin didáctico del aprendizaje de las bases de CSS.

En éste se pueden encontrar información básica sobre el uso de CSS, así como pequeños proyectos: Calculadora básica, Galería GRID, Cronómetro, etc.

Tecnologías usadas: HTML, CSS, JavaScript

CALCULADORA

Proyecto sencillo de una calculadora con operaciones aritméticas básicas.

Contiene una serie de botones ordenados mediante HTML y CSS, programados con eventos y POO con JavaScript.

Tecnologías usadas: HTML, CSS, Javascript

CRONÓMETRO

Proyecto sencillo de un crónometro para uso personal.

Contiene interfaz sencilla con íconos por botones programados con eventos con JavaScript.

Tecnologías usadas: HTML, CSS, JavaScript

GALERÍA GRID

Proyecto sencillo de una galería de imágenes, con propósitos autodidactas del uso de la propiedad GRID.

Contiene una serie de imágenes como representación de una galería, junto con eventos programados en JavaScript para la ampliación de las mismas.

Tecnologías usadas: HTML, CSS, JavaScript
